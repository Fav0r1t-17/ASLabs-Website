# TextNormalizer

**Clean, normalize, and format messy text quickly.**

TextNormalizer is a lightweight Windows desktop utility for cleaning and transforming messy text copied from AI chats, documents, logs, notes, and web content.

It is designed for practical everyday text cleanup: removing duplicates, fixing spacing, stripping HTML, extracting useful data, and preparing text for further editing or processing.

TextNormalizer runs locally on your machine and does not require an account or subscription.

---

## Features

### Text cleaning

- Remove duplicate lines
- Remove empty lines
- Collapse multiple blank lines
- Trim leading and trailing whitespace
- Normalize bullet styles
- Clean common formatting noise
- Remove repeated filler-style text where supported

### Text transformation

- Convert case: UPPERCASE, lowercase, Title Case, Sentence case
- Sort lines alphabetically
- Reverse line order
- Add line numbers

### Data extraction

- Extract URLs
- Extract email addresses

### Special utilities

- Remove HTML tags
- Auto-clean pipeline for quick processing

### Interface

- Simple desktop interface
- Dark mode
- Drag-and-drop file loading
- Preview before applying changes

## Why TextNormalizer

Copied text often contains duplicates, broken formatting, empty lines, HTML fragments, inconsistent bullets, and other cleanup issues.

TextNormalizer helps make that text usable faster.
