# AS Labs - Official Website

Welcome to the official repository for **aslabs.dev**, the home of **AS Labs**.

AS Labs builds simple, local-first AI tools focused on real-world use. We prioritize simplicity, local control, and practical results.

## Products

### 1. TextNormalizer
A lightweight desktop tool that cleans, normalizes, and transforms messy text.
*   **Key features:** Remove duplicates/empty lines, extract URLs/emails, case conversion, and automated cleanup of AI-generated filler text.
*   **Workflow:** Local-first, offline, no cloud, no account, one-time purchase.

### 2. W-Bee
*Coming soon.* W-Bee is a desktop application designed to transform learning materials into structured, study-ready documents.

## Tech Stack
*   **Frontend:** HTML5, CSS3, Vanilla JavaScript.
*   **Hosting:** Static site hosted via [Porkbun Static Hosting](https://porkbun.com).
*   **Architecture:** Static, bilingual (BG/EN), responsive desktop-first design.

## Project Structure
- `index.html`: Main site structure, navigation, and content sections.
- `style.css`: Unified styles for responsive layout and theme support.
- `script.js`: Language switching, smooth scrolling, and form handling.
